Dique-Sense

Presentamos el sistema de monitoreo para el nivel del Canal del Dique “Dique-Sense” en el municipio de Santa Lucia, que provee un sistema de alerta y prevención que permite medir el nivel del brazo fluvial tomando datos en intervalos de 1 hora, posibilitando visualizar el nivel en tiempo real y consultar históricos.

VISUALIZAR MEDICIONES

Objetivos

Objetivo general

Diseñar e implementar un punto de monitoreo remoto para el nivel de un brazo fluvial que permita la visualización y gestión de alarmas desde una plataforma web.

Objetivos especificos

  • Seleccionar la plataforma para sensado remoto del nivel de un cuerpo de agua con las características del Canal del Dique.
  • Adaptar la plataforma para sensado remoto para la implementación en la intemperie
  • Diseñar una interfaz web que permita visualizar los datos de la medición y compararlos con niveles críticos de una forma intuitiva para el acceso del público en general.
  • Validar experimentalmente la operación del sistema y evaluar su desempeño en condiciones ambientales tanto normales como en contingencias.

Funcionamiento

Se construyó un módulo en la orilla del canal del dique, el cual está constituido por un microcontrolador ESP32 y un sensor JSN-SR04T. Este módulo se encarga de tomar datos cada hora y enviarlos al servidor web para almacenarlos en bases de datos. La información recolectada se podrá observar mediante una página web en la cual se utilizó el lenguaje de programación Python para el desarrollo del Back-End, para la parte del Front-End se utilizó HTML, Javascript y un framework de CSS llamado Bootstrap con el fin de agilizar el proceso de desarrollo de la página. Para el almacenamiento de los datos, se creó una base de datos usando MySQL y se utilizó MySQL Workbench que es una herramienta visual que permite la administración, diseño, gestión y mantenimiento de bases de datos.
